    Cómo Jugar

    1. Cada jugador ingresa su nombre.
    2. Se selecciona una pregunta al azar para un jugador.
    3. El jugador tiene un tiempo limitado para responder.
    4. Si responde correctamente, gana un punto. Si no, se resta un punto.
    5. Si un jugador llega a 0 puntos, queda descalificado.
